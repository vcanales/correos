## correos-cli

Utilidad de CLI para obtener informacion sobre envios de Correos de Chile a partir de su codigo de siguimiento.

### Instalacion

```sh
npm i -g correos-cli
```

### Uso

```sh
correos <codigo_de_seguimiento>
```
